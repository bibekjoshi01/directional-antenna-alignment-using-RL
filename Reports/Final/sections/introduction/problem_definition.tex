\subsection{Problem Definition}

Existing reinforcement learning approaches for antenna alignment mainly target mmWave
beamforming systems, antenna arrays, or other high-computation settings, leaving a
clear gap for lightweight embedded implementation using only RSSI
feedback. At the same time, classical RSSI-based methods such as exhaustive scanning
and hill-climbing are simple to implement but remain insufficient for this problem:
exhaustive scanning is too slow for responsive operation, while hill-climbing is easily
degraded by noise, multipath effects, and local optima. As a result, there is still no
clear low-cost solution that combines embedded reinforcement learning, RSSI-only
sensing, and practical mechanical antenna steering on a resource-constrained platform.

This project addresses that gap by defining the problem as real-time alignment of a
mechanically steerable directional antenna at the receiver using only packet-level RSSI
measurements and internally tracked antenna orientation. The controller must run on an
ESP32-S3 with limited computation, memory, and power, without relying on phased
arrays, GPS, angle-of-arrival estimation, or detailed propagation models. Within these
constraints, the work investigates whether a tabular reinforcement learning policy can
be deployed on embedded hardware, validated in real-world operation against simulation
expectations.
